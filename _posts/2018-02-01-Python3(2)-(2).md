## while문

4

## for문


* for 문에 사용될 수 있는 `시퀀스형 자료`로는 `문자열`, `리스트`, `튜풀`, `사전`과 같은 `내장 클래스`와 <br> `내장 함수 iter()`로 생성되는 `이터레이터 객체`, 또는 이러한 동작이 가능하도록 관련 연산자를 재정의한 <br> `(__iter__(), __next__() )객체`이다.
``` python
>>> l = [10, 20, 30]
>>> iterator = iter(l)
>>> for i in iterator:
...     print(i)

10
20
30
```

* 2중 for문도 가능하다.

``` python
>>> for n in [1,2]:
...     print(" -- {0} 단 --".format(n))
...     for i in [1,2,3]:
...             print(" {0} * {1} = {2}".format(n,i, n*i))

 -- 1 단 --
 1 * 1 = 1
 1 * 2 = 2
 1 * 3 = 3
 -- 2 단 --
 2 * 1 = 2
 2 * 2 = 4
 2 * 3 = 6
```
