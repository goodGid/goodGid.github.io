# AVL 트리

* `AVL : 높이 균형 이진 트리`
    * 트리 전체를 재 균형 시키지 않고도 트리의 균형 유지
    * 삽입, 삭제 연산 시간이 짧음
    * 검색시간 : O(logN)
    * 왼쪽 / 오른쪽 서브트리의 높이 차 <= 1
    * AVL 트리는 `메인 메모리`에 거주하는 내부 구조로 <br> 트리 크기가 너무 커서 메인 메모리에 구축할 수 없을 경우 <br> --> 균형 m-원 탐색 트리
