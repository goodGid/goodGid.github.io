## 2차 코딩 테스트 후기

* 일시 : 18. 09. 29 (토)

* 시간 : 120분

* 문제 : 3문제

* 장소 : 역삼 YBM

* 환경 : 프로그래머스 웹 버전 

* 후기

    - 펜 종이 사용 X
    
    - 오로지 컴퓨터로만 작업 가능
    
    - 코드 저장 기능만 있음 
    
    - 제출 기능 X
    
    - STL 사용 가능 
    
    - 코딩 테스트때는 텀블러 사용 가능
    
    - 인적성때는 텀블러 사용 가능 X
    
    - 프로그래머스 환경인데 main함수가 있음
    
    - main부터 시작해야하함

    - 문제가 그렇게 쉽진 않았다.

    - 자료 구조를 활용한 문제가 나올거라 생각했지만 틀렸다.

    - 그냥 **완전 탐색**이였다.
