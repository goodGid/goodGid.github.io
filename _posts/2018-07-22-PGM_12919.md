## [1] Answer Code (18. 07. 22)

``` cpp

string solution(vector<string> seoul) {
    string answer = "김서방은 ";
    int size = seoul.size();
    for(int i=0; i<size; i++){
        if(seoul[i] == "Kim")
            answer += to_string(i);
    }
    answer +="에 있다";
    return answer;
}

```
