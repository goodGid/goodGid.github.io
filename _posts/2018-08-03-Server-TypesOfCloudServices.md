# 클라우드 서비스의 종류

## IaaS (Infrastructure as a Service)

* IaaS는 메모리, CPU, Storage 등과 같은 인프라를 제공하는 형태의 클라우드 서비스

* AWS의 EC2, MS의 Azure

<br>

## PaaS (Platform as a Service)

* IaaS의 확장판

* Application을 가동하기 위한 플랫폼을 제공하는 클라우드 서비스

<br>

## SaaS (Software as a Service)

* S/W를 제공하는 형태의 클라우드 서비스
