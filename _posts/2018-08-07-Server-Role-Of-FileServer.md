# 파일 서버의 역할

* 파일 서버는 LAN, WAN과 같은 네트워크에서 다른 컴퓨터와 파일을 공유 or 데이터를 송수신하기 위해 사용하는 서버이다.

* 파일 서버를 사용하면 여러 클라이언트가 하나의 파일을 다룰 때 발생하기 쉬운 `갱신 무결성 문제`를 막을 수 있다.

* 파일 서버에서 중요한 설정은 바로 `액세스 권한`이다.
