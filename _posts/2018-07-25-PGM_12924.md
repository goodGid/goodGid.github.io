## [1] Answer Code (18. 07. 25)

``` cpp

int solution(int n) {
    int answer = 0;
    int a,b;
    a=b=1;
    
    int v=1;
    while (a <= n) {
        if(v == n){
            answer ++;
            v += ++b;
        }else if( v < n){
            v += ++b;
        }else{
            v -= a++;
        }
    }
    return answer;
}

```
