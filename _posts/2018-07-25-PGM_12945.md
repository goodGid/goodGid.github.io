## [1] Answer Code (18. 07. 25)

``` cpp

long long dp[100001] = {0};

long long solution(int n) {
    long long answer = 0;
    int mod = 1234567;
    
    dp[0] = 0;
    dp[1] = 1;
    for(int i=2; i<=100000; i++){
        dp[i] = dp[i-1] + dp[i-2];
        dp[i] %= mod;
    }
    
    answer = dp[n] % mod;
    
    return answer;
}

```

---

### [1] Answer Code (18. 07. 25)

* dp[i] %= mod; 이부분을 빼먹어서 틀렸었다.
