## Answer Code [18. 02. 10]
``` cpp

#include <iostream>
#define ll long long
using namespace std;

int main() {
    ll a,b,v;
    cin >> a >> b >> v;
    ll c = a - b;
    
    ll l=1,r=1e9;
//    r = v / c;  // [1]
    
    ll ans;
    while (l <= r) {
        ll mid = (l+r) >> 1; // [2]
        ll value = ( mid-1 ) * c + a;
        
        if( value >= v ){
            ans = mid;
            r = mid - 1;
        }
        else{
            l = mid + 1;
        }
    }
    cout << ans << endl;
    
    return 0;
}


```

### [2]

int mid로 선언을 하면

l = max_int - 1

r = max_int일 때 

mid는 int형을 벗어나기 때문에 터진다 !
