## ConvNet의 활용예


![](/assets/img/machine_learning/ML_11_4_1.png)

 

Q. Output값 어떻게 나온거지?
{: .notice}

```
[ 227 x 227 ]을 [ 11 x 11 ] 사이즈로 
stride : 4 값으로 
필터를 하게 되면
4n+7이라는 식이 나온다.
여기서 n값을 구하면 된다.
4n+7 = 227 ==> n = 55
```


![](/assets/img/machine_learning/ML_11_4_2.png)

 


Q. Output값 어떻게 나온거지?
{: .notice}

```
[ 55 x 55 ]을 [ 3 x 3 ] 사이즈로 
stride : 2 값으로 
필터를 하게 되면
2n+1이라는 식이 나온다.
여기서 n값을 구하면 된다.
2n+1 = 55 ==> n = 27
```
