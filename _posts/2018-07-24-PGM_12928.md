## [1] Answer Code (18. 07. 24)

``` cpp

// [1]
int solution(int n) {
    int answer = 0;
    for(int i=1; i<=n; i++){
        if( n % i == 0){
            answer += i;
        }
    }
    return answer;
}

// [2]
int solution(int n) {
  int nSum=0;
  for (int i=1; i<=sqrt(n); i++)
  {
    if (n%i == 0)
    {
       nSum += (i + n/i);
    }
  }
    return nSum;
}


```
