## [1] Answer Code (18. 08. 01)

{% highlight cpp %}

string solution(string s) {
    s.at(0) = toupper(s.at(0));
    for (int i = 1; i < s.length(); i++)
        s.at(i) = (s.at(i - 1) == ' ') ? toupper(s.at(i)) : tolower(s.at(i));
    return s;
}
{% endhighlight %}

---

### [1] Answer Code (18. 08. 01)

* ' ' 공백이 여러개 들어갈 수 있다는걸 뒤늦게 깨달았다.
