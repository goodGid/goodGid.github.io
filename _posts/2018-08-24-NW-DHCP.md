# DHCP(Dynamic Host Configuration Protocol)

* 네트워크 안의 컴퓨터에 대해 네트워크 정보(IP즈소,서브넷 마스크)를 자동으로 설장하기 위한 프로토콜이다.

* DHCP를 이용하는 네트워크에서는 이러한 관리를 모두 DHCP 서버가 대신하여 IP 주소의 중복과 같은 인위적 실수가 발생하지 않는다.
