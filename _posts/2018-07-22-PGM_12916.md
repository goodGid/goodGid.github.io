## Problem 
Problem URL : **[문자열 내 p와 y의 개수](https://programmers.co.kr/learn/courses/30/lessons/12916)**

---

## [1] Answer Code (18. 07. 22)

``` cpp

bool solution(string s){
    bool answer = false;
    int size = s.size();
    int y_cnt = 0;
    int p_cnt = 0;
    
    for(int i=0; i<size; i++){
        if(s[i] == 'p' || s[i] == 'P'){
            p_cnt ++;
        }
        else if(s[i] == 'y' || s[i] == 'Y'){
            y_cnt++;
        }
    }
    
    if( y_cnt == p_cnt)
        answer = true;
    
    return answer;
}

```
