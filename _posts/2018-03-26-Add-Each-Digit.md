## Related Problem

Problem URL : **[분해합](https://www.acmicpc.net/problem/2231)**

---

``` cpp

int cal(int num) {
    int sum = 0;
    while (num!=0) {
        sum += num % 10;
        num /= 10;
    }
    return sum;
}

/*
input : 123
output : 6


input : 51515
output : 17

*/
 
```
