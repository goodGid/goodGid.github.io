## [1] Answer Code (18. 07. 25)

``` cpp

bool solution(string s){
    bool answer = true;
    
    int cnt = 0;
    for(int i=0; i<s.length(); i++){
        s[i] == '(' ? cnt ++ : cnt --;
        if(cnt < 0) break;
    }
    
    if(cnt != 0)
        answer = false;
    
    return answer;
}


```
